# 🔴 RedHawk - Advanced Scanner for PenetrationTesting

## 🦅 Overview

**RedHawk** is a comprehensive penetration testing framework designed for security professionals and ethical hackers. It provides powerful tools for web application security testing, network reconnaissance, and vulnerability assessment.

## ✨ Features

### 🌐 Web Scanner (20+ Vulnerabilities)
- SQL Injection (Error-based, Time-based, Union-based)
- XSS (Reflected, Stored)
- LFI / RFI
- Command Injection
- Open Redirect
- CSRF Detection
- Information Disclosure
- Security Headers Analysis
- Directory Bruteforce
- Backup File Exposure
- Admin Panel Detection
- Cookie Security Analysis
- NoSQL Injection
- SSTI (Server-Side Template Injection)
- XXE (XML External Entity)
- SSRF (Server-Side Request Forgery)
- LDAP Injection
- File Upload Bypass
- Host Header Injection

### 🖧 Network Scanner
- Port Scanning
- Service Detection

### 📡 Reconnaissance
- Subdomain Enumeration

### 📊 Reporting
- HTML (Beautiful web report)
- TXT (Plain text)
- JSON (Raw data)
- PDF (Professional document)
